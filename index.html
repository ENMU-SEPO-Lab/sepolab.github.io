The SEPO Lab

Software Engineering and Performance Optimization

Our current research focuses on software engineering and computer science education, with a strong emphasis on integrating real-world, industry-aligned practices into the classroom. Our work explores how automated and AI-driven tools—such as CI-based code assessment, static analysis, natural language processing techniques, and large language models—can support formative feedback, improve code quality, and enhance learning outcomes for novice programmers. We also investigate memory architecture design and innovative data management techniques aimed at improving application performance.
